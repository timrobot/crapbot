## Libraries

  - pyserial (arduino)
  - pygame (ps4)
  - opencv-python (camera)
  - https://www.jetsonhacks.com/2019/12/22/install-realsense-camera-in-5-minutes-jetson-nano/

## Compile Libraries prior to running

```
mkdir build && cd build
cmake .. && make
cd ..
```

## Entry point

```
sudo python3 multiprocess_runnable.py
```
